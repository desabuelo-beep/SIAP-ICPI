# SIAP-ICPI v1.0 — Sistema de Integridad Algorítmica Preventiva

**Plataforma GovTech QUADRUM | GAD Municipal de Montecristi | Ecuador**

## ¿Qué es el SIAP-ICPI?

El SIAP-ICPI es un sistema de verificación algorítmica preventiva que mide 
la **Congruencia Programática e Intersistémica** entre lo que un gobierno 
local planifica, ejecuta y reporta públicamente.

Detecta la **brecha entre el ICM oficial (auto-reporte SIGAD) y el ICPI 
verificado algorítmicamente** — en Montecristi 2024: **58.76 puntos porcentuales**.

## Arquitectura del PMV

| Componente | Descripción |
|---|---|
| **Motor Central** | H01–H26: 28 hojas, 797 fórmulas activas |
| **Módulo de Salidas** | H27–H31: 5 interfaces de solo lectura |
| **Silos de datos** | 8 fuentes: CNE, PDOT, POA, SERCOP, LOTAIP, eSIGEF, SIGAD, CPCCS |

## Índices calculados

- **ICPI Global**: 41.24% → Transición Crítica (escala AVEP)
- **ICM SIGAD oficial**: 100%
- **Brecha ICM-ICPI**: 58.76 pp → Señal MOM-I activa
- **ITAM**: 62.1% (Transparencia Activa Municipal)
- **IFE**: 85%+ (Fidelidad Electoral — 20 promesas CNE)
- **IED Obras Públicas**: 45.5%
